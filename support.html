What SwiftBuilder Pro Does

  • Create iOS app UIs visually without writing code.
  • Preview iPhone UI layouts on macOS (no physical iPhone required).
  • Build multi-screen app flows with navigation and tabs.
  • Generate/export SwiftUI source files for Xcode and Swift Playgrounds.

Feature Scope

  • 20 templates and 22 visual elements.
  • Shell modes: Single, Navigation, Tabs.
  • SF Symbols integration with searchable catalog.
  • Local save/load workspace support.

Purchases

  • Pro plans: monthly and yearly subscriptions.
  • Restore Purchases restores active entitlements.
  • Manage Subscription opens Apple subscription management.

Troubleshooting

  • No StoreKit products found: in development, attach a local .storekit config in Run Scheme. In production, verify App Store Connect product setup.
  • Restore has no visible change: ensure active subscription exists under the same Apple ID.
  • Export mismatch concerns: retest with current app version and provide template name plus screenshots when reporting.

Include in Bug Reports

  • App version and macOS version.
  • Template name and selected shell mode.
  • Expected vs actual behavior.
  • Screenshots and exported code snippet.
