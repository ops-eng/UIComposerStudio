# SwiftBuilderPro

SwiftBuilderPro is a macOS visual builder for designing SwiftUI iOS app interfaces and exporting project-ready Swift files.

## Overview

SwiftBuilderPro is designed for both non-coders and developers:
- Build real app screens visually.
- Preview inside iPhone device frames directly on macOS.
- Configure multi-screen navigation flows.
- Export generated SwiftUI code to continue in Xcode or Swift Playgrounds.

## Key Value

- No coding required to start building app interfaces.
- No physical iPhone required for visual preview workflow.
- Fast iteration with templates, layers, and inspector controls.
- Production-friendly Swift file export path.

## Main Features

### Projects and Workspace
- Create projects from visual templates.
- Set app name and organization used in generated files.
- Save/load local workspace snapshots.
- Continue editing across sessions.

- Layer tree for hierarchy management.

  - Foreground/background colors
