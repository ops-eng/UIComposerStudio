UIComposerStudio Help

Visual SwiftUI app builder for macOS

Reference notes: UIComposerStudio is a visual SwiftUI builder for creating iOS app UIs on macOS. It supports project-style code generation, local workspace save/load, screen flow setup, tab shell preview, and Swift export that you can continue in Xcode or Swift Playgrounds.
Most important: You can create Swift iOS app code without writing code and without owning an iPhone. Build visually on your Mac, preview in device frames, then copy or export Swift files.

Main Features

Projects

  • Create and iterate app projects with visual editing.
  • Set app name and organization for generated code headers.
  • Save/load workspace locally to continue where you left off.
  • Reset project from selected template.

Screens

  • Add, rename, select, and remove screens.
  • Set screen destinations for button navigation.
  • Model real multi-screen app flows before export.

Navigation Types

  • Single: one root screen.
  • Navigation: push/pop destination flow.
  • Tabs: root tabs for multi-section apps.

Layers + Inspector

  • Layer tree for fast selection and hierarchy control.
  • Drag/drop reorder in the layer panel.
  • Inspector controls for spacing, alignment, colors, typography, corner radius, opacity, and element-specific options.

Preview, Devices, and Output

Device Preview

  • Build visually in iPhone mockups directly on macOS.
  • Supported devices: iPhone SE, iPhone 16, iPhone 16 Pro, iPhone 16 Pro Max, iPhone 17e, iPhone 17, iPhone 17 Pro, iPhone 17 Pro Max, iPhone Air.
  • Light and dark appearance toggle in preview.

SF Symbols

  • Core library: 112 symbols.
  • Search and apply symbols from the inspector.
  • Free tier uses a subset; Pro unlocks full set.

Code Output

  • Show Code: open generated Swift source in-app.
  • Copy Code: copy generated Swift output to clipboard.
  • Export Project: export generated `.swift` files for Xcode or Swift Playgrounds.

Free vs Pro Snapshot

Free: core templates, core elements, 1 preview device (iPhone 16), up to 3 screens, local save/load. Pro: all templates, all elements, all device presets, full symbols set, unlimited screens, project export.
